# 🪝 Hook 三位一体 · 终极定义

```
第一重: 钩 (Extract)    — 30条暗行通道·海底捞针
第二重: 化 (Transform)  — 理解设计哲学·提取规律·化为知识
第三重: 创 (Create)     — 跨源融合·自我创造·生成原创新设计
```

## 第三重：创 — 终极能力

```
这不是"复制A+B"。
这是"理解A+理解B → 创造C"。

创的能力:
  ① 跨源融合    — 取A的配色+B的字体+C的布局 = 全新设计
  ② 风格迁移    — "把Stripe变成暗色主题"·"把Apple变得更有活力"
  ③ 智能变体    — 一个设计自动生成3个变体（保守·平衡·激进）
  ④ 空白生成    — "我要一个科技感·大标题·深色·有粒子效果" → 自动生成
  ⑤ 自我进化    — 每次创建都在学习·下次创建更好
```

## 规则·借字哲学

```
借 = 不是偷。是学习。

① 借其神·不借其形
   学Apple的极简哲学·不是复制Apple的Logo
   学Stripe的商业逻辑·不是复制Stripe的文案
   学Linear的排版节奏·不是复制Linear的字体

② 化其魂·化为己用
   理解了WHY → 知道WHEN用 → 创造HOW
   不是"Stripe用了48px标题" → 是"金融类产品适合大标题建立信任"

③ 创其新·超越原版
   融合Apple+Stripe+Linear → 一个全新的·更好的设计
   不是三选一·是三生万物

借 = 站在巨人肩膀上·看得比巨人更远。
```

## Hook 完整形态

```
🪝 Hook = 钩 + 化 + 创

输入:  "我要一个金融科技感·暗色·精英气质的落地页"

过程:
  钩 → 知识库调取 Apple(配色)·Stripe(布局)·Linear(字体)

       + 完整的CSS·动画·响应式
       + 设计说明: 为什么这么设计
```
