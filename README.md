## 快速开始

```bash
# 安装
npm i -g butterfly-hook

# 钩取
butterfly hook https://apple.com

# 对比
butterfly hook https://apple.com --compare=https://stripe.com

# 融合创造
butterfly create --sources=apple.com,stripe.com,linear.app --intent=tech
```
