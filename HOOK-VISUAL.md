# 🎣 Hook · 可视化产品方案

## 为什么是Web应用

```
Desktop App  → 需要下载·更新·平台适配
Mobile App   → 屏幕太小·设计分析不方便
CLI          → 开发者用·普通用户不会
Web App      → 零安装·任何设备·可分享·可盈利

hook.design = 最佳形态
```

## 产品形态：Web + 插件 = 双引擎

```
🎣 Hook = Web Dashboard (主场) + Chrome插件 (随身钓竿)

  主场: hook.design → 深度分析·图库·对比·创造
  随身: Chrome插件  → 浏览时一键钩取·即时洞察
```

## Chrome插件（随身钓竿）

```
浏览任何网页时:
  → 点击Hook图标
  → 侧边栏滑出
  → "这个页面的设计DNA:"
  → 配色·字体·间距·一键钩取
  → "打开完整报告" → 跳转hook.design

不离开当前页面·即时洞察
```

## 技术栈

```
前端:  Next.js 16 + Tailwind v4 + shadcn/ui
后端:  Node.js (复用Hook脚本)
数据库: PostgreSQL + Redis
部署:   Vercel (前端) + Railway (后端)
存储:   Cloudflare R2
```
