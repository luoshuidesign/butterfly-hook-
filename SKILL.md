# 🦋 /butterfly-hook · 蝶变钩子

> **Hook = 钩子。精准、锋利、一击即中。**
> 钩住任意网站 → 拉取设计DNA → 化为己用。
>
> 不是爬虫。不是截图。是"钩"——像鹰爪一样抓住猎物，不松口。

## 为什么叫 "Hook"？

```
Webhook    → 钩住Web事件
Git Hook   → 钩住Git动作
React Hook → 钩住组件状态
🦋 Hook    → 钩住网站设计

Hook = 抓住·拉取·化为己用
不是"克隆"(clone) — 那是复制品
不是"提取"(extract) — 那是分离
是"钩"(Hook) — 鹰爪入肉，不松口，不遗漏
```

## 使用

```bash
# 一键钩取
butterfly hook https://apple.com

# 深度钩取（10维DNA）
butterfly hook https://apple.com --deep

# 钩取+对比
butterfly hook https://apple.com --compare=http://localhost:3000

# 钩取+生成React组件
butterfly hook https://apple.com --output=react
```
